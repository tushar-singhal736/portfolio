# Portfolio Kaise Chalayein

## Error: "Missing script: start" ?

Ye tab aata hai jab aap **galat folder** se `npm start` chala rahe ho.

---

## Sahi tarika

### Option 1: Terminal se (Cursor / VS Code)

1. Terminal kholo: **Ctrl + `** (backtick)
2. Pehle **myapp** folder mein jao:
   ```
   cd myapp
   ```
3. Phir start karo:
   ```
   npm start
   ```

### Option 2: Seedha myapp folder kholna

1. Cursor mein **File → Open Folder**
2. Ye folder select karo: `react learning\myapp`
3. Jab myapp hi root folder ho, tab terminal mein sirf likho:
   ```
   npm start
   ```
